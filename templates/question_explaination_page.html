MATH

Question Type Explanation Examples
Arithmetic Questions in this category require some piece of basic arithmetic to be performed (e.g. addition, subtraction, multiplication, division). What is the difference in height between Mt. Everest and the Empire State Building?
Standard Deviation Questions in this category require the standard deviation to be calculated over a set of data points. What is the standard deviation among the age of students at Northwestern?
Correlation Questions in this category require a correlation to be calculated over a set of data points. Is there a positive correlation between the amount of chocolate consumed and a personâ€™s happiness level?
Average Questions in this category require an average value to be calculated over a collection of values or data points. What is the average population of all countries in the world?
Median Questions in this category require the median value to be found in a collection of values. What is the median population of countries in the world?
Mode Questions in this category require the most common value to be found in a collection of values. What is the most common birthday among these people?
Counting Questions in this category require that a collection of items be counted in order to answer the question. How many countries are there in the world?
Range Questions in this category require the range of values to be found in a collection of values What is the range of number of burritos a college student eats?

RETRIEVAL

Question Type Explanation Examples
Qualitative Property Retrieval Questions in this category require a single property to be retrieved about a given entity. Who was Eleanor Roosevelt married to?
Numeric Retrieval Questions in this category require a numeric value to be retrieved. What is the population of Earth?
Boolean Retrieval Questions in this category require a boolean value to be retrieved or produced given a set of properties or text regarding an entity. Is George Washington French?
Datetime Retrieval Questions in this category require a datetime value to be retrieved. When was Barack Obama born?
Set Retrieval Questions in this category require a set of values to be retrieved What are all the ben and jerrys flavors?
Qualitative Property Multihop Retrieval Questions that require a sequence of simple retrieval questions to be answered. When was the person Eleanor Roosevelt is married to born?

COMPARISON

Question Type Explanation Examples
Numeric Comparison Questions in this category require multiple numeric values to be found and compared in order to determine which is greater, lesser, equal, etc. Is the Sears Tower taller than the Empire State Building?
Datetime Comparison Questions in this category require multiple datetime values to be found and compared in order to determine which is first, last, the same, etc. Was George Lucas born before Steven Spielberg?
Qualitative Comparison Questions in this category require qualitative properties of entities to be found and compared. Is the sky the same color as grass?
Superlative Questions in this category require that multiple values be found and ranked in order to determine the top value. Which book has the highest average user rating?
Set Intersection Questions in this category require that at least two sets of values be found in order to perform an intersection operation and find the common elements. What movie directors are both taller than six feet and from California?
Set Union Questions in this category require that at least two sets of values be found in order to perform a union operation and merge the sets together into one. Which films were filmed in either California or Georgia?
Set Difference Questions in this category require that at least two sets of values be found in order to find the elements that are in one set, but not in the other sets. Which films were created in California, but not Georgia?
Mathematical Comparison Questions in this category can be viewed as inheriting from Metric Comparison and Mathematics as they require both a mathematical computation to be performed, as well as a comparison of the computed values. By how many feet is Denali taller than Mt. Rainier?
Set Property Satisfaction Questions in this category require that a set of entities that satisfy a specific property are found. What buildings have a height greater than 500 feet?

BOOLEAN

Question Type Explanation Examples
Boolean And Questions in this category require a boolean AND expression to be evaluated from a collection of boolean values. Are Serena Williams and Roger Federer both American?
Boolean Or Questions in this category require a boolean OR expression to be evaluated from a collection of boolean values. Is either Serena Williams or Roger Federer American?

DEFINITIONAL

Question Type Explanation Examples
Definitional Questions in this category require that the definition or meaning of an entity or word is provided to the user. What is the definition of fire?

OPEN ENDED

Question Type Explanation Examples
Causal Explanation Questions in this category require some kind of description of why an event has occurred or why something is the way it is. How did World War I start?
Opinion Questions in this category require an opinion. What do you think of my hair?
